### To setup:
> npm install
> npx tsc

### Commands:

For help:
>npm run-script help

Async/Await example:
>npm run-script asyncAwait

Callback example:
>npm run-script callback

Apicall async example:
>npm run-script apiAsync

Apicall promise example:
>npm run-script apiPromise

Calculator:
>npm run-script calculator
